# E-commerce Manager System

## 🚀 What it does
This project simulates a backend system for managing e-commerce products, including updating prices and stock.

## 🧠 Features
- View product list
- Update product prices
- Update product stock
- Simulates real backend operations

## 🛠 Tech Used
- Python

## 💡 Why I built this
To demonstrate how e-commerce systems manage product data and handle updates efficiently.

## ▶️ How to run

1. Install Python
2. Run:
   python app.py
